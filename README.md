Quote and Color Generator

A fun web application built with vanilla JavaScript, HTML, and CSS that displays random quotes and changes the background color each time you click a button. 
This project involves the basics of practice JavaScript DOM manipulation and event handling.

Features:

Generates a random motivational quote on button click

Changes the background color dynamically

Clean and simple user interface styled with CSS

Works on both desktop and mobile

Purpose:
This project was created to practice JavaScript events, DOM manipulation, and adding interactive elements to a webpage.
